# インタラクティブなポートフォリオサイト

## コンセプト
- 「ドット絵」×「インタラクティブ（シェーダー）」をテーマにしたポートフォリオ。
- ゲーム開発の経験（GLSL/HLSL）を活かし、背景や作品展示にリッチなシェーダー表現を取り入れる。

## ページ構成・コンテンツ
1. **Home**
    - 名前、アイコン、ハンドルネーム
    - 技術スタック (Skills)
    - 経歴 (Career/About)
2. **2DWorks**
    - 普段制作しているドット絵作品のギャラリー
    - 画像のクリックで拡大表示などの機能
3. **ShaderWorks**
    - シェーダー作品の展示
    - 映像やGIFだけでなく、マウス操作等でインタラクティブに動くWebGLのライブデモも掲載

## インタラクション要件
- **背景シェーダー**: ドットの格子状のデザインをベースにし、マウスカーソルが合わさった部分が反応するエフェクト、またはGLSLによる流体/粒子(パーティクル)表現を背景に描画する。
- **UIとの連携**: React側で取得したマウス座標やスクロール量を `uniforms` としてGLSLシェーダーに渡し、リアルタイムに絵作りを変化させる。

## 技術スタック (確定)
- **フレームワーク**: React (構築ツール: Vite)
- **シェーダー・WebGL描画**: React Three Fiber (Three.js の React用ラッパー) + Drei
- **ルーティング**: React Router (複数ページの切り替え)
- **スタイリング**: Vanilla CSS または Tailwind CSS (ドット絵のレトロな雰囲気や、プレーンなUIを作成)
